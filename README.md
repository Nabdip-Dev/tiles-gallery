## 🎯 Overview

**Tileshop** is a sleek and responsive tile gallery web application designed to deliver a smooth browsing experience.  
Users can explore curated tile collections, view detailed information, and manage their profiles—all within a modern, animated interface.

---

## ✨ Key Features

- 🔐 Secure Authentication (Login & Registration)
- 👤 User Profile Management (Update Info)
- 🧱 Dynamic Tile Gallery with Search
- 🔍 Detailed Tile View Page
- 🔒 Protected Routes for Authorized Users
- 🎨 Clean & Modern UI Design
- 🎬 Smooth Animations (Animate.css)
- 📱 Fully Responsive (Mobile + Desktop)

---

## 🧭 Routes

### 🌐 Public Routes
- `/` → Home  
- `/all-tiles` → Explore Tiles  
- `/login` → Login  
- `/register` → Create Account  

### 🔐 Private Routes
- `/tile/[id]` → Tile Details  
- `/my-profile` → User Profile  

---

## 🛠️ Tech Stack

- ⚡ Next.js (App Router)
- 🎨 Tailwind CSS + DaisyUI
- 🔐 BetterAuth (Authentication)
- 🗄️ MongoDB (Database)
- 📡 JSON Server (Mock API)
- 🎬 Animate.css
